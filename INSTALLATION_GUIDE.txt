KETC Website Version 5 – Installation Guide

1. Extract this ZIP file first. Do not upload the ZIP file itself.
2. Open your GitHub repository: kabirengineering/kabirengineering.github.io
3. Click Add file > Upload files.
4. Drag all extracted contents together:
   - assets folder
   - index.html
   - style.css
   - robots.txt
   - sitemap.xml
   - CHANGELOG.txt
   - INSTALLATION_GUIDE.txt
5. Confirm GitHub shows assets/... files, not loose SVG files at root.
6. Click Commit changes directly to the main branch.
7. Wait 1–3 minutes.
8. Open https://kabirengineering.com and press Ctrl+F5.

Important: The proposal form currently uses a mailto fallback. For a fully professional online form with document upload, connect Formspree, Web3Forms, or Google Forms in the next contact/SEO phase.
