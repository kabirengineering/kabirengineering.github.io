KETC Website V3 Phase 2 - Installation Guide

1. Keep your current live website backup safe.
2. Upload the contents of this folder to the GitHub repository root.
3. Make sure the assets folder remains as /assets/.
4. Commit directly to the main branch.
5. Wait 1-3 minutes for GitHub Pages deployment.
6. Visit https://kabirengineering.com and press Ctrl+F5.

Important: Do not upload the ZIP file itself. Upload the extracted files and the assets folder.
