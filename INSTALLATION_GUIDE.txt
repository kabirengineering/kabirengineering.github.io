KETC Website V4 - Installation Guide

1. Extract this ZIP file.
2. Open the extracted folder.
3. Upload the assets folder and all root files to GitHub Pages repository.
4. Keep the folder structure exactly as provided.
5. Commit directly to the main branch.
6. Wait 2-5 minutes for GitHub Pages deployment.
7. Refresh https://kabirengineering.com using Ctrl + F5.
