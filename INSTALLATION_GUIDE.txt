KETC Website V3 Phase 3 - Installation Guide

1. Open GitHub repository: kabirengineering/kabirengineering.github.io
2. Click Add file > Upload files.
3. Upload the extracted folder contents, not the ZIP file.
4. Upload these together:
   - assets folder
   - index.html
   - style.css
   - robots.txt
   - sitemap.xml
   - CHANGELOG.txt
   - INSTALLATION_GUIDE.txt
5. Confirm GitHub shows files such as /assets/favicon.svg, /assets/hero-engineering.svg, etc.
6. Commit directly to the main branch.
7. Wait 1-3 minutes and open https://kabirengineering.com
8. Press Ctrl+F5 if the old version appears.
